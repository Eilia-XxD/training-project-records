### 基本操作

1. 初始化仓库，并把项目克隆到本地仓库

```
git init 
git clone 项目链接
```

然后就可以愉快的添加内容啦~

2. 提交内容

   ```
   git add . 			（这里的点代表添加全部内容）
   git status			（查看变动，可省略此步）
   git commit -m ""     （引号内填写简要说明）
   git push			（推送）
   ```
